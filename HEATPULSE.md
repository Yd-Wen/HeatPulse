## 需求

作为一个 AI 编程学习者，需要第一时间获取一些热点（如 AI 大模型更新），我不能依赖人工手动搜素，而是利用工具自动发现指定的热点变化，或者发现最新的热点，并且及时给我发送通知，让我第一时间了解。

由于是一个小项目，所以打算采样较为敏捷的开发方式，不用过多地人工介入和工程化。

### 功能

具体来说，几个功能点：

- 用户手动输入要监控的关键词，当这个关键词相关的内容真的出现（注意利用 AI 识别假冒内容），第一时间发送通知

- 每隔一段时间，自动搜集用户输入的指定范围（比如 “AI 编程” ）内的热点，并且能够让用户看到

### 产品形式

- 响应式兼容的 web 页面

- 封装成 Agent Skills 技能，能够交给其他的 AI Agent 来监控和发现热点

## 环境准备

1. VS Code

2. Claude Code

3. 大模型

先用 deepseek

4. 扩展（比如 MCP）

参考文章 https://ai.codefather.cn/library/2013071149038157826 中的 AI 扩展工具文章

需要安装 MCP：

什么是 MCP，简单来说就是让 AI 获取外部工具和数据源，增强 AI 能力，参考文章 https://ai.codefather.cn/library/2010974694076256258

- Filecrawl MCP 网页内容获取，便于 AI 读取网页内容、判断热点信息 
- Context7 获取最新文章，防止 AI 使用过时的 API

需要安装 Agent Skills:

什么是 Skills，简单来说就是 AI 能够学习使用各种专业技能，而不用每次都重复输入提示词、编写脚本等，参考https://ai.codefather.cn/library/2013072104018264066

- UI UX Pro Max 用来美化前端页面
- Find Skills 用来安装需要的技能
- Skil Creator 用来制作技能。因为我们需要把工具封装成 Agent Skills，所以可以用它来开发规范的技能，让各种 AI Agent 能识别和使用

## 方案设计

由于项目不复杂（只是个工具），不写专业提示词，直接让 AI 根据需求设计方案。

```markdown
你是一位专业的程序员，现在请你根据需求，帮我设计方案、人工确认、分步骤完成开发、执行测试、找我验收。
```
