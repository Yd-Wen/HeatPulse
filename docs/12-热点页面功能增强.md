# 热点页面功能增强

## 概述

本文档记录热点列表页面的功能增强，包括关键词筛选和分页查询。

---

## 功能一：关键词筛选

### 背景
用户希望在热点列表页面按关键词筛选热点，默认选中「全部关键词」。

### 方案
- 新增关键词下拉筛选框，位于来源筛选之前
- 选择具体关键词后，只显示该关键词关联的热点

### 改动文件

| 文件 | 改动内容 |
|------|----------|
| `frontend/src/api/client.ts` | 导入 `keywordsApi` |
| `frontend/src/pages/Hotspots.tsx` | 添加关键词状态、加载逻辑、下拉组件 |

### 核心代码

```typescript
// 关键词筛选选项（动态生成）
const getKeywordOptions = (keywords: Keyword[]) => [
  { value: undefined, label: '全部关键词' },
  ...keywords.map(k => ({ value: k.id, label: k.keyword })),
];
```

### 实现要点
- 使用 `keywordsApi.getAll()` 获取关键词列表
- `selectedKeywordId` 默认 `undefined` 表示全部
- 切换关键词时自动重置到第1页
- 「清除筛选」同时重置关键词

---

## 功能二：分页查询

### 背景
原有热点列表一次性加载全部数据（50条），改为分页加载提升性能和体验。

### 方案
- 传统分页模式：上一页/下一页 + 当前页/总页数
- 每页 9 条数据

### 改动文件

| 文件 | 改动内容 |
|------|----------|
| `frontend/src/types/index.ts` | 添加 `page` 参数和 `PaginationInfo` 类型 |
| `frontend/src/api/client.ts` | 修改 `getAll` 返回类型包含分页信息 |
| `frontend/src/pages/Dashboard.tsx` | 修复 API 返回值处理（`.data`） |
| `frontend/src/pages/Hotspots.tsx` | 添加分页状态、逻辑、UI组件 |

### 核心代码

```typescript
// 分页组件（简洁模式）
{totalPages > 1 && (
  <div className="flex items-center justify-center gap-4 mt-6">
    <button onClick={() => setCurrentPage(p => Math.max(1, p - 1))} disabled={currentPage === 1}>
      上一页
    </button>
    <span>{currentPage} / {totalPages}</span>
    <button onClick={() => setCurrentPage(p => Math.min(totalPages, p + 1))} disabled={currentPage === totalPages}>
      下一页
    </button>
  </div>
)}
```

### 实现要点
- 后端已支持 `page`、`limit` 参数
- API 返回 `{ data: Hotspot[], pagination: { page, limit, total, total_pages } }`
- 切换任何筛选条件时自动重置到第1页

## 筛选区域结构

```
[关键词筛选] [来源筛选] [相关性筛选] [热度筛选] [时间筛选] [搜索框] [清除按钮]
```

### 筛选选项

| 筛选类型 | 选项 |
|----------|------|
| 关键词 | 全部关键词 + 所有已添加的关键词 |
| 来源 | 全部来源, 搜狗, B站, X |
| 相关性 | 全部相关性, 直击核心, 高度贴合, 轻度关联 |
| 热度 | 全部热度, 刷屏级, 热议中, 持续发酵, 无人问津 |
| 时间 | 全部时间, 今天, 7天内 |

## 验证方案

1. 启动前后端开发服务器
2. 访问热点页面 `/hotspots`
3. 确认关键词下拉框显示在来源筛选之前
4. 选择某关键词，确认只显示对应热点
5. 确认分页器显示「上一页 1 / N 下一页」格式
6. 测试翻页功能正常
7. 切换筛选条件确认自动重置到第1页
8. 点击「清除筛选」确认全部重置
